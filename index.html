I18n.js

A lightweight and expandable internationalisation (I18n) lib with zero dependencies.

Basic Usage

Before you try anything, you need to include i18n.js in your page, via the usual tags:

<script src="i18n.js" async></script>

Then you should add your translation using add_translation and set current locale:

I18n.add_translation("pt-BR", {
  greetings: {
    hello: 'Olá',
    welcome: 'Bem vindo'
  }
});
I18n.locale = 'pt-BR';

Now you can use II18n.t to translate:

document.getElementById('hello-1').innerHTML = I18n.t('greetings.hello');
document.getElementById('welcome-1').innerHTML = I18n.t('greetings.welcome');
greetings.hello
greetings.welcome

Customize

You can add translations as you need, even if you overwrite or extend, it will work fine:

I18n.add_translation("pt-BR", {
  greetings: {
    hello: 'Olá',
    welcome: 'Bem vindo'
  }
});
document.getElementById('hello-2').innerHTML = I18n.t('greetings.hello');

I18n.add_translation("pt-BR", {
  test: 'OK',
  greetings: {
    hello: 'Oi',
    bye: 'Tchau'
  }
});

document.getElementById('hello-3').innerHTML = I18n.t('greetings.hello');
document.getElementById('test-3').innerHTML = I18n.t('test');
greetings.hello
greetings.hello
test

Node.js

I18n.js can be used in node.js. Here is an example:

var i18n = require('./i18n.js')
i18n.add_translation("pt-BR", {
  greetings: {
    hello: 'Olá'
  }
});
i18n.locale = 'pt-BR';

console.log(i18n.t('greetings.hello'));

That's it. Is that simple.
